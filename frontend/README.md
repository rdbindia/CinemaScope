# CinemaScope Project

## Project Overview

Cinema Scope is an application designed to analyze trends in the industry to better understand box office performance.

### Pre-requisites

- docker
- docker-compose
- PHP
- Laravel
- Node.js and npm
- MySQL database

`git clone git@github.com:wildalaskan/skeleton-app-vue.git`
